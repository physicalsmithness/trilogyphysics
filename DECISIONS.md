**d002 (Architecture and Housing combined; Authoring separate; split housing only on trigger).** One chat owns both project shape and the engine/display housing, plus one or more Authoring chats. Rationale: mirrors the deliberate Fields choice, which works, and keeping schema and display in one head is the cheaper default for a single-engine project.
