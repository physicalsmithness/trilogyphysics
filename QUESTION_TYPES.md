# Trilogy Physics: QUESTION_TYPES (to be filled at M1)

The IB side had documents summarising all the question types, and that summary was the starting point for authoring. Trilogy has had no such document. Building it is the Architecture chat's first M1 task, because authoring without it tends to drift narrow.

This file is a scaffold. Fill it from the extracted bank (the Codex DB and `outputs/previews/`) and the examiner reports, not from memory.

## What to produce

A map of every question type AQA Trilogy physics actually uses, with, for each:

- A name and a short description.
- The mark range it appears in.
- Whether it is calculation, written explanation, recall, MCQ/tick-box, practical-method, or graph/diagram.
- The mark-scheme shape (how marks are split and gated), drawing on the Codex six-layer tagging for calculation types.
- Typical misconceptions and examiner-report pain points.
- Whether the current engine can grade it, or whether it needs a NEW_QTYPE.

## Starting checklist (from the pilot tagging and the Y10 exam work, expand and verify)

- Single-formula calculation (recall vs from-insert; with and without prefix conversion).
- Multi-stage calculation (chained formulae; the electricity multi-route mark schemes; bridges between quantities).
- Calculation with unit/prefix/standard-form/sig-fig marks (the "give the unit" mark that sits last; sig-figs embedded in evaluation; standard-form gating).
- Anti-ECF-gated calculation (AQA's "this mark only if equation X used", roughly once per Higher paper).
- Graph technique (read-off, gradient, area, inverse-proportion reasoning).
- Short written explanation (2 to 4 marks).
- Level-of-response 6-marker (the heavy explanation item; thinking-distance/braking-distance, RP method descriptions, etc).
- Required-practical method and analysis (RP14 SHC, RP15 resistance, RP19 F=ma, and the rest).
- Recall and definition (single-mark identify, "what is meant by", momentum-conserved-type definitions).
- MCQ / tick-box (single and multi-select).
- Cross-tier shared items (the same question as Foundation Q05/06 and Higher Q01).

Cross-reference the existing Magnetism bank (about 160 questions) against this map when folding it in at M2, to see which types it already covers and which are thin.
